Cross-device sync settings

Manually-entered odds are saved in this browser's local storage only. To sync them across devices, paste a GitHub Personal Access Token here. When you save odds, they will be written directly to data/history.json in the repo and will appear on all devices after the next workflow run (~30 min).

Create a fine-grained PAT at github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Grant Contents: Read and Write on the simplejack2/NRFI repository only. The token is stored in this browser's local storage and never sent anywhere except the GitHub API.
